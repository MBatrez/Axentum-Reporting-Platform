# Axentum Reporting Platform

An open-source multi-platform ad reporting platform built for performance marketing agencies in LATAM.

## About

This project is being developed by [Axentum Media](https://axentumedia.com), a digital advertising consultancy based in Mexico. The goal is to provide a lightweight, self-hostable reporting solution that consolidates data from Meta Ads, Google Ads, and other platforms — accessible to agencies that can't afford enterprise tools.

## Planned Features

- Multi-platform campaign data consolidation (Meta Ads, Google Ads, TikTok Ads)
- Customizable client-facing dashboards
- Automated reporting workflows
- White-label ready for agencies

## Status

🚧 **Active development** — Built with Claude Code as part of an AI-assisted development workflow.

## Tech Stack

- Vanilla JS / HTML / CSS
- Custom PHP backend
- Meta Ads API, Google Ads API
